How to play

Choose a mode, then guess the color shown in the swatch by typing the correct value or picking the right answer.

  • Easy mode: select from four choices.
  • Hard mode: type the exact answer.
  • HEX format: #FF0000 or ff0000.
  • RGB format: rgb(255,0,0) or 255 0 0.
  • HSL format: hsl(0,100%,50%) or 0 100 50.
